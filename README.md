# 📊 HR Analytics Dashboard

## 🔍 Project Overview

This project focuses on analyzing employee data to uncover key insights related to workforce performance, attrition, and organizational trends.
The goal is to support data-driven decision-making using interactive dashboards and structured data analysis.

---

## 🎯 Objectives

* Analyze employee demographics and job roles
* Identify factors contributing to employee attrition
* Monitor key HR metrics and KPIs
* Provide actionable insights for business improvement

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* SQL
* Power BI 
* Data Visualization Techniques

## 📊 Key Insights

* High attrition observed in specific job roles and departments
* Salary and job satisfaction have a strong impact on employee retention
* Work-life balance plays a crucial role in employee performance
* Certain experience levels show higher turnover rates

---

## 📈 Dashboard Features

* Interactive filters and slicers
* KPI indicators for quick insights
* Department-wise and role-wise analysis
* Attrition and performance visualization


---

## ▶️ How to Use

1. Download or clone the repository
2. Open the dashboard file:

   * Power BI → Open `.pbix` file
   * Excel → Open `.xlsx` file
  
3. Explore the dashboard and interact with filters
